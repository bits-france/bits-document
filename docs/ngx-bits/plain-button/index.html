🔘 Plain Button

Composant de bouton d'action hautement configurable avec support d'icônes, de spinners de chargement, d'états désactivés et d'effets ripple d'Angular Material.

Sélecteur HTML : ngx-bits-plain-button

📸 Aperçu Visuel​

🚀 Utilisation​

1. Importation​

Importez le composant standalone dans votre composant Angular :

import { NgxBitsPlainButtonComponent } from '@bits-solution-org/ngx-bits';

@Component({
  imports: [NgxBitsPlainButtonComponent],
  // ...
})

2. Template HTML​

Liez la configuration du bouton via la propriété buttonContainer :

<ngx-bits-plain-button [buttonContainer]="myButtonConfig" />

📊 Référence API​

Inputs​

PropriétéTypeDescription
buttonContainer *PlainButtonInterfaceObjet de configuration décrivant le comportement et le style du bouton.

Outputs​

ÉvénementTypeDescription
clickEventEventEmitter<void>Émis lorsque le bouton est cliqué (uniquement s'il n'est ni désactivé ni en chargement).

Méthodes publiques​

Vous pouvez appeler ces méthodes à l'aide d'un @ViewChild :

MéthodeSignatureDescription
setButton()(buttonContainer: PlainButtonInterface): voidMet à jour dynamiquement toute la configuration du bouton.
setLoading()(loading: boolean): voidActive ou désactive l'état de chargement (affiche un spinner).
setDisable()(disable: boolean): voidActive ou désactive le bouton.

📝 Configuration (PlainButtonInterface)​

ChampTypeDéfautDescription
title *string—Le libellé textuel affiché sur le bouton.
clickFunction() => void—Fonction callback exécutée lors du clic.
isDisabledbooleanfalseDésactive le bouton si true.
isLoadingbooleanfalseAffiche le spinner de chargement si true.
actionType'button' | 'submit''button'Attribut HTML type du bouton.
iconOrImageButtonIconOrImage—Configuration de l'icône/image optionnelle (Material icon ou URL).
