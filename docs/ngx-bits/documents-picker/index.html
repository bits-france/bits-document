📎 Documents Picker

Composant d'upload de fichiers complet, gérant le glisser-déposer (drag-and-drop), le multi-téléchargement et la liste des fichiers sélectionnés avec bouton de suppression.

Sélecteur HTML : ngx-bits-documents-picker

📸 Aperçu Visuel​

Zone de Dépôt standard​

Rendu d'un Fichier Sélectionné​

🚀 Utilisation​

Standalone​

1. Importation​

import { NgxBitsDocumentsPickerComponents } from '@bits-solution-org/ngx-bits';

@Component({
  imports: [NgxBitsDocumentsPickerComponents],
  // ...
})

2. Template HTML​

<ngx-bits-documents-picker 
  [multiple]="true" 
  [name]="'Mes Documents'" 
  (changeFileArrayEvent)="onFilesChange($event)" 
/>

📊 Référence API​

Inputs​

PropriétéTypeDéfautDescription
multiplebooleanfalsePermet la sélection de plusieurs fichiers simultanément si true.
namestring'Document'Label ou placeholder affiché dans le sélecteur.

Outputs​

ÉvénementTypeDescription
changeFileArrayEventEventEmitter<File[]>Émis à chaque fois qu'un fichier est ajouté ou retiré du sélecteur. Renvoie la liste complète des fichiers sélectionnés.
